# PTV Tracker Backend

This is the Node.js/Express backend for the Public Transport Victoria Tracker app.

## Features
- API integration with PTV
- Serves transport data to frontend

## Setup
1. Install dependencies: `npm install`
2. Start server: `npm start`

## PTV API Key
- Register for a PTV API key at https://www.ptv.vic.gov.au/
- Add your API key to a `.env` file in this folder:
  ```
  PTV_API_KEY=your_key_here
  ```
